# Ham-Diary App (Android)

> ### 소개
귀여운 햄스터 감정 아이콘과 함께 하루를 기록할 수 있는 간단한 일기장 어플입니다.   
기능은 다음과 같습니다.
1. 일기 생성, 삭제, 조회
2. 햄스터 아이콘으로 감정 표현
3. 날짜별 감정 기록 조회
